Undergrad
Capstone Project

This project was used to update Washington State Parks trail data and to provide users with a map of existing trails. The data was collected by GPS and corrected with LiDAR imagery to provide a highly accurate map that can be printed for users. A georeferenced PDF of this map was also created to give users a map that can be downloaded to a mobile device to be used with apps like Avenza Maps.
